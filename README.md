# 🛒 E-Commerce Shop Project
### 📝 Project Description
This project is a fully functional web application built as the final assessment for the **Frontend Development** course at **International Digital University (IDU)**. It is a modern e-commerce platform where users can browse product catalogs, filter items by categories, and manage a shopping cart. The application is designed with a component-based architecture and is fully responsive across desktop, tablet, and mobile devices.
### 🛠 Tech Stack
The project utilizes the following technologies as per the assignment requirements:
***React 18+ (Vite):** Core UI library and fast build tool.
***Tailwind CSS:** For utility-first, responsive styling.
***React Router v6:** Handles navigation between the home, product, and cart pages.
***Axios:** Used to fetch real-time product data from the **Fake Store API**.
***React Context API:** Manages global state for the shopping cart (adding/removing items).
***React Icons:** Provides essential UI icons for the cart and search features.

### 🚀 How to Run Locally
1. Clone the repository:
```bash
git clone https://github.com/moonlight0102/ecommerce-shop.git
```
2. Install dependencies:
```bash
npm install
```
3. Start the development server:
```bash
npm run dev
```
